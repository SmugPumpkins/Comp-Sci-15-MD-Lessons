# Challenge - Physics Calculator
The object of this challenge is to practice adding event bindings to a Tkinter program.

* There is a labeled input field for the `time_1` it took the object to travel that far
* There is a labeled input field for the `distance_2` an object has travelled
* There is a labeled input field for the `time_2` it took the object to travel that far

* A button to calculate `acceleration`
* A label to display the calculated `acceleration`
